# network-architecture-enterprise
Diseño de arquitectura de red para una empresa con una sede principal y dos sucursales.  Herramientas:  Cisco Packet Tracer  Características:  Configuración de routers y switches Conectividad entre sedes Simulación de entorno real  Autor: Johann Salvador Rodriguez

Este proyecto consiste en el diseño e implementación de una arquitectura de red empresarial que conecta una sede principal con dos sucursales, simulando un entorno real de comunicación organizacional.

La solución fue desarrollada utilizando Cisco Packet Tracer, permitiendo validar la conectividad y el funcionamiento de la red mediante pruebas prácticas.

Objetivos:

Diseñar una red escalable y funcional para múltiples sedes
Garantizar la conectividad entre todas las ubicaciones
Simular un entorno real de infraestructura de red
Aplicar conceptos de direccionamiento IP y configuración de dispositivos

Tecnologías y herramientas:

Cisco Packet Tracer
Configuración de routers y switches
Direccionamiento IP
Simulación de redes

Funcionalidades principales:

Interconexión entre sede principal y sucursales
Configuración de dispositivos de red (routers, switches, PCs)
Validación de conectividad mediante pruebas de red (ping)
Simulación de flujo de datos entre diferentes nodos

Pruebas realizadas:

Se realizaron pruebas de conectividad para asegurar el correcto funcionamiento de la red:

Comunicación entre dispositivos dentro de la misma sede
Comunicación entre diferentes sedes
Verificación de rutas de red
